### Prerequisites
* Node version
  ```sh
  14.x
  ```
* Npm version
  ```sh
  6.x
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ClawGibson/NutriSo.git
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Run
   ```sh
   npm run dev
   ```
